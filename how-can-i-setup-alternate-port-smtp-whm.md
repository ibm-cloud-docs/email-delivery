# Define an alternate port for SMTP in WHM
{: #define-alt-port-smtp-whm}
{: faq}

1. Log in to **WHM**.
2. Click **Service Manager**.
3. Set the port after **Run another copy of Exim on port**.
4. Click **Save**.
